# Landing Page — bepartner.lat

## Que es
Landing page publica de Bepartner. Sitio de venta con caso de exito, comparativa, y formulario de calificacion que conecta con Calendly. Objetivo: convertir ecommerce que ya venden en llamadas de auditoria gratuita.

## Stack
- Next.js 14 (App Router)
- Tailwind CSS
- Framer Motion
- Lucide React (iconos)
- React Hook Form (formularios)
- Inter + Plus Jakarta Sans (fonts)
- Deploy: Docker multi-stage en Easypanel (puerto 3000)

## Reglas de diseno
1. Fondo SIEMPRE #f8f7f5 (blanco tiza). Nunca blanco puro.
2. Color principal: #753D94 (violeta).
3. Cards: blanco #FFFFFF, rounded-xl, shadow-sm, border border-gray-100.
4. Headlines: Plus Jakarta Sans. Texto general: Inter.
5. Espanol rioplatense: "vos" → "Escala", "Sabes", "Invertis".
6. NO mostrar precios en la landing.
7. Secciones oscuras: #1a1a2e con gradient.

## Regla de posicionamiento
NUNCA usar "agencia" ni "consultor" para referirnos a nosotros. Bepartner es "socio de crecimiento" o "sistema de escalado". La palabra "agencia" SOLO se usa para referirse a la competencia.
